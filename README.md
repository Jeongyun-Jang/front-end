## 기능 소개
1)테스트를 통한 사용자 맥주 타입 및 맥주 2종 추천 기능
2)맥주별 맛, 판매처, 후기 정보 한곳에서 비교하기 기능
3)마셔본 맥주의 후기를 작성하여 다른사람과 공유하는 기능
4)전체 맥주의 후기를 실시간 피드로 확인하는 기능
5)마이비어에서 자신이 작성한 도감 및 좋아요한 맥주 목록 모아보기 기능
6)맥주 및 관리자에게 건의하기 기능

## 기술 소개
